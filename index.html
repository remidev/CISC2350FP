Project Overview

The bulk of the work on this site focuses on two main areas:

  • Projects Page — Features three interactive Perlin noise visualizations built with Vanilla JavaScript, p5.js, and three.js
  • Portfolio Section — A recreation of a professional cinematographer's Squarespace portfolio with custom styling and CDN-hosted media
    • View the original Squarespace portfolio
Note about the Portfolio: While the inner portfolio section closely resembles the original Squarespace design, not all links and interactions are fully functional. To return to the main site from the portfolio, use the "← Back to CISC2350FP" link in the footer at the bottom of any portfolio page.
